# Skills

A collection of AI skills for developers, designed to be model-agnostic and focused on best practices.

## Available Skills

### [Dev App Java](./dev-app-java/SKILL.md)
End-to-end Java application development (creation, structuring, refactoring, quality review) for Vanilla Java, Spring Boot, Quarkus, or Struts. Use it whenever you need to write Java applications, generate projects, apply best practices (Javadoc, unit tests, design patterns), or guide architecture and code quality.

### [DB Persistence & SQL](./db-persistence-sql/SKILL.md)
Design and optimization of the persistence layer (PostgreSQL, JPA/Hibernate, Migrations). Focus on the 'Repository' part of the Controller/Service/Repository pattern, ensuring data integrity, high performance, and clean migrations.
